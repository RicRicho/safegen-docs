# SafeGen

SafeGen is public documentation for a privacy-preserving child-safety and age-assurance concept from Ric Richardson's invention portfolio.

The public model is simple: parents can verify that a child controls a mobile number, and authorised platforms or regulators can use a child-number registry to reduce under-age access without requiring face scans, identity documents, or broad surveillance.

## Public positioning

- Parent-driven child-number verification.
- Platform/accountability layer for age-restricted services.
- Privacy-first design: no biometrics, no content access, and minimal public disclosure.
- Intended to complement Australia's eSafety age-restricted material framework and comparable international platform-accountability regimes.
- Patent-backed at category level; detailed claim language and implementation mechanisms are intentionally withheld from this public repository.

## What this repository is

This repository is a clean public documentation surface for SafeGen: problem framing, architecture-level notes, commercial context, and public review material.

It is not a private data room, investor pack, negotiation record, or implementation repository.
