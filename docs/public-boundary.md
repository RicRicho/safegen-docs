# Public Boundary

This repository may contain public problem framing, architecture-level summaries, commercial positioning, and non-confidential review notes.

It must not contain:

- patent claim text, application numbers, or specification detail unless explicitly cleared;
- registry, cryptographic, HSM, query, platform-detection, or implementation mechanisms;
- live outreach, government, investor, legal, or negotiation records;
- unrelated private data-room material;
- credentials, private contact lists, or internal operating notes.
